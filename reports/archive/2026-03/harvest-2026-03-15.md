## Skill 候選分析

### Strong — 強烈建議建立

#### 1. `sqlite-to-postgres` — SQLite 到 PostgreSQL 遷移

- **用途**: 自動化從 SQLite 遷移到 PostgreSQL/Supabase 的完整流程
- **觸發**: 使用者要求遷移資料庫引擎、提到 Supabase、或 `.env` 出現 `DATABASE_URL` 指向 PostgreSQL 但程式碼仍用 SQLite 語法
- **建議分類**: `backend`
- **理由**: 本次花費 5+ 個 session 完成此流程，涉及：
  1. 掃描 SQLite 專用語法（`julianday`, `INSERT OR IGNORE`, `strftime`, `PRAGMA`, `executescript`）
  2. 生成 PostgreSQL 相容 schema（`SERIAL` vs `INTEGER PRIMARY KEY`, `JSONB` vs `TEXT`, `NOW()` vs `datetime('now')`）
  3. 撰寫資料遷移腳本（型別轉換、JSON 欄位處理）
  4. 更新 connection layer（`sqlite3` → `psycopg2`）
  5. 驗證所有 API endpoint 正常運作
- **與現有 `db-migration` 差異**: `db-migration` 專注於 schema 版本管理工具（Alembic/Knex），不涵蓋跨引擎遷移

#### 2. `launchd-agent` — macOS LaunchAgent 管理

- **用途**: 建立、修改、除錯 macOS launchd agent（plist 檔案、排程、狀態查詢）
- **觸發**: 使用者要求建立排程任務、提到 plist/launchctl/LaunchAgent、或需要在 macOS 上設定 cron-like 自動化
- **建議分類**: `workflow`
- **理由**: 跨多個 session 反覆出現的模式：
  - 讀取/解析 plist 檔案（`plistlib`）
  - 使用 `launchctl list` 檢查狀態
  - 產生新 plist（`StartCalendarInterval`、`StandardOutPath`、`EnvironmentVariables`）
  - 用 `launchctl load/unload` 管理生命週期
  - 排程欄位格式轉換（cron 語法 ↔ plist 格式）
- **無現有 skill 覆蓋此領域**

### Moderate — 值得考慮

#### 3. `codebase-storage-audit` — 專案資料儲存全面盤點

- **用途**: 自動掃描專案中所有資料儲存機制（DB、檔案上傳、JSON/CSV、.env 設定）
- **觸發**: 使用者要求了解專案資料流、準備遷移、或進行架構審查
- **建議分類**: `quality`
- **理由**: sales-assistant 遷移前有 2 個 Explore agent 做了深度盤點，模式固定：
  1. 搜尋 `*.db`, `*.sqlite`, `*.sql`, `.env*` 檔案
  2. 讀取 connection/config 模組
  3. Grep 所有 `import sqlite3/psycopg2/mongoose` 等 DB driver
  4. 盤點 `uploads/`, `data/`, `storage/` 目錄
  5. 檢查 schema 檔案與實際使用是否一致
- **但**: 此流程可能太專案特定，且 Explore agent 已能勝任

#### 4. `streamlit-dashboard` — Streamlit 儀表板腳手架

- **用途**: 快速搭建 Streamlit 多頁面應用（app.py + pages/ + lib/ + SQLite 資料層）
- **觸發**: 使用者要求建立管理介面、儀表板、或提到 Streamlit
- **建議分類**: `frontend`
- **理由**: sk-dashboard 的建構過程是一個完整模板：
  - `app.py` 主框架 + `st.sidebar.radio` 導航
  - `lib/db.py` SQLite 封裝
  - `pages/*.py` 模組化頁面
  - `requirements.txt` 依賴管理
  - `.streamlit/config.toml` 主題設定
- **但**: 此模式與 `frontend-design` 有部分重疊，且 Streamlit 並非通用需求
