## Skill 候選

### 🟢 Strong：`react-fab-search-pattern`

| 欄位 | 內容 |
|------|------|
| **Purpose** | 一鍵長出「typed 後端 search endpoint + debounced FAB + keyboard nav（↑↓ Enter ESC）+ route 跳轉」全套組件 |
| **Trigger** | 使用者說「加一個浮動搜尋」「FAB 搜公司/ticker 跳頁」「全域快速搜尋」「command palette 雛形」 |
| **Category** | `frontend/` |
| **Rationale** | Session 2 花了 ~80+ 訊息把這個從 0 做到能用，pattern 高度可複用（CompanySearchFAB → 任何 entity 搜尋）。現有 `frontend-design` 是通用設計、`ui-ux-pro-max` 不涵蓋這個具體 widget。寫成 skill 可省下 schema → endpoint → typed client → debounced component → mount 的 5 步重複勞動 |
